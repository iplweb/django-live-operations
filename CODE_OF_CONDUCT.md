# Code of Conduct

This project adopts the [Contributor Covenant](https://www.contributor-covenant.org/),
version 2.1.

In short: be respectful and constructive. Harassment and discriminatory
behaviour are not tolerated. Report concerns to the maintainers at
support@iplweb.pl.

The full text is available at
<https://www.contributor-covenant.org/version/2/1/code_of_conduct/>.
